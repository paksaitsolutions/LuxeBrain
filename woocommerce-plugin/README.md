# LuxeBrain AI for WooCommerce

**Copyright © 2024 Paksa IT Solutions**

AI-powered recommendations, dynamic pricing, and automation for WooCommerce fashion stores.

## Installation

1. Download `luxebrain-ai.zip`
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin
3. Upload the zip file and activate
4. Go to LuxeBrain AI settings and enter your API key

## Features

- AI Product Recommendations
- Dynamic Pricing
- Customer Segmentation
- Demand Forecasting
- Marketing Automation

## Requirements

- WordPress 5.8+
- WooCommerce 5.0+
- PHP 7.4+
